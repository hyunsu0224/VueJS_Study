Title : Second
< v-bind : title에 관하여 >
여기에 나오는 「v-bind」속성은 '디렉티브'라고 한다
v- 라는 접두어가 있으면 렌더링된 DOM에 의해 특수한 반응형 동작을한다.(jquery의 이벤트기능)
지금 사용하는 title 이라는 속성은 참고하는 현재는 message라는 Vue인스턴스를 최신상태로 유지시킨다. title -> 마우스를 오버해서 값이 출력되도록 한다. popup 이벤트와 비슷

콘솔창에 app2.message = "입력값"을 넣으면 입력값이 출력된다.
내 위에 마우스를 올리면 동적으로 바인딩된 title을 볼수있다.
Title : Third
< v-if에 관하여(true/false) >
「v-if」속성은 텍스쳐 혹은 지정한 요소의 관해서 true 값인경우 보여지게하고 false 값인 경우 보여지지 않도록한다. 텍스트뿐만 아니라 DOM의 구조에도 데이터를 바인딩 할 수 있다. Vue 요소가 삽입 업데이트 제거 될때 트랜지션 효과를 줄 수 있다.

「v-for」 디렉티브는 배열의 데이터를 바인딩하여 Todo 목록을 표시한다.
