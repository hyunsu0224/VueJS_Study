데이터 바인딩은 element의 class list와 inline-style을 조작하기 위해 일반적으로 사용
이 두 속성은 v-bind를 사용하여 처리할 수 있다.
우리는 표현식으로 최종 문자열을 계산하면 된다.
그러나 문자열 연결에 간섭하는 것은 오류가 발생하기 쉽다.
이러한 이유로, Vue는 class와 style에 v-bind를 사용할 때 특별히 향상된 기능을 제공한다.
표현식은 문자열 이외에 객체 또는 배열을 이용할 수 있다.
v-bind는 jquery의 attr과 같은 느낌으로 v-bind:class면 class를 변경한다는 의미가 된다 js 내용으로
